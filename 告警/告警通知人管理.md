```

```

## 删除

```
DELETE GET /admin/alert/v1/notifier/:id
```

## 获取公众号信息

> 用于显示公众号二维码, 绑定微信前需要先让用户关注公众号
```
GET /admin/wechat/v1/app/info
```

## 绑定微信

```
GET /public/v1/wechat/auth/start?router=bindAlertNotifier&alertNotifierId=1&appId=wx3e1ed87cfac336e6
```

> 参数:

    <tr>
