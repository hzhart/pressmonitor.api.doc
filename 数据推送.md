# 数据推送平台接口协议

## 适合范围
-------------------------
` 本API提供推送数据至第三方平台`


## 请求语法
-------------------------
`
HttpMethod / HTTP/1.1
Host: $host
Content-Type: application/json;
`

## 支持的认证方式
-------------------------
`
HTTP Basic Auth
`

## 推送数据
-------------------------

> 推送数据到第三方平台

```
POST /url(自定义)
```
